# Mutiny
A Python library emulating the game logic behind Chris Brown's 
[Coup server](https://github.com/octachrome/treason), for use with
[Ray's](https://github.com/ray-project/ray) MultiAgentEnv.

# Installation

In the main directory:

`pip install -e .`
